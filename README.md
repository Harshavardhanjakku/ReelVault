**Saved Reels App**

A simple web application that helps users save, search, and manage Instagram reel links or notes in one place.
This project is built using **HTML, CSS, JavaScript, and Firebase Firestore**.

**Features**
- Save reel links or notes
- Search saved reels
- Delete (unsave) reels
- Cloud storage using Firebase Firestore
- Clean Instagram-style UI

**Tech Stack**
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Firebase (Firestore Database)

**Idea Behind This Project**

I wanted to create a simple tool to organize saved Instagram reels because Instagram does not provide a good search system for saved content. This app solves that by allowing users to store and search their saved reels easily.

**How to Run**
1. Clone the repository
2. Open `index.html` in browser
3. Add your Firebase config if needed
4. Start saving reels
